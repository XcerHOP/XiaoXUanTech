# Xiaoxuan Tech — Official Website

Official website for **Foshan Xiaoxuan Technology Co., Ltd.**

## Quick Start

Open `index.html` directly in a browser, or run a local server:

```bash
# Python 3
python3 -m http.server 8080

# Then visit http://localhost:8080
```

## Structure

- `index.html` — Main page (all English)
- `styles.css` — AI-tech × Chinese aesthetic styling
- `main.js` — Particle network, animations, interactions
